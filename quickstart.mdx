## How to get started?

1. Register your account at [https://dashboard.zyvik.com](https://dashboard.zyvik.com)

2. Buy credits on [https://dashboard.zyvik.com/credits](https://dashboard.zyvik.com/credits)

3. Create a new API key on [https://dashboard.zyvik.com/keys](https://dashboard.zyvik.com/keys)

Now you can use Zyvik.
